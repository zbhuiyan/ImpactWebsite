Formal Usability Study

From the very beginning, we debated the effectiveness of listing tasks in timeline versus in calendar format. This is the most integral part of our app and it needs to be streamlined. In our last prototype, we implemented a timeline, as shown below:

We will use the formal usability study to evaluate the effectiveness of each approach. To do this, we will use the following quantitative metrics:

  • Average time to find and open a given task. This will be the driving metric, as it will provide insight into how visible tasks are.
  • Number of clicks to open a given task.
  • Average distance into the future that users tend to click. We are curious as to which format encourages more long term planning versus short term planning.

Additionally, we will consider qualitative feedback from the users on what they found appealing or useful about each format (and if there are any clarity issues or things that bother them about either design).

Ideally, we would hope to test at least twenty users in order to provide accurate averages for our numerical data. Each user should only try one design. If they were to try both, they would be more familiar with the app by the second test, and would be faster during these tests. With enough users, randomly assigning users to each design will provide accurate measurements.

However, considering our time limits, it’s unlikely that we will have more than three user tests. We will therefore ask each tester to do both tests. In the case of a very slow or a very fast user, this will produce less of an outlier. In order to account for familiarity with the app, we can separate tests by a few days (i.e. ask a user to test one design one day, and the other a few days later). This way, what they know about using the app will have faded a bit because they will have had some time to forget about that interaction.

We expect users to find a given task quicker and with fewer clicks with the timeline layout, since tasks will be more visible in this format. However, we also expect the timeline layout to be less encouraging of long term planning.
